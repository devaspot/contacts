Personal Data Sync.

Windows Sync Client

Synrc Contacts Application for Vista is about organizing, import/export, syncing your Address Books with Google Contacts, Microsoft Outlook, NOKIA Phones, Windows Contacts local folder, LDAP Directory, Yahoo! Contacts and Windows Live Contacts.

Download for Windows

Built on Windows Azure, Live Mesh and Microsoft .NET

Features

  • Puritan and Minimalistic interface
  • Symmetric synchronization (no master and slave)
  • Original Sync Replication Protocol
  • No data loss, duplicating or inconsistance
  • Designed for Multicore CPUs
  • Open architecture
  • No passwords stored
  • Instant full cross-field search
  • Supports Windows Sync Center

Puritan & Minimalistic interface

Synrc Contacts is a "buddhist view" zen Application for Vista and Windows 7. Its simplified one-button design makes your life easy in terms of managing personal contact information.


Symmetric synchronization

You do not need specify what sync source is master or slave, which one will be rewrited or not. The symmetric sync algorithm guarants you that no data loss, contacts duplicating or data inconsistance accurse. The application starts and loads all contacts from Windows Contacts folder by default. That sync source sets up as default. Every sync session performs with the default sync source.

Instant full cross-fields search

You can find contacts by any data stored within vCARD: phone, e-mail, name, postal address, category, etc. Results are filtered instantly by your typing.

Designed with multithreading in mind

The sync Engine uses 7 threads to fetch, synchronize and update sync source. That guarants you that your synchronization will be finished as soon as possible and defines high responsiveness and smoothness of application.


Security Issues

No passwords are stored on your computer from your sync sources. Passwords can be remembered only within a life of the application. When the application shutdowns you have to retype passwords. This guarants you that no password leek is possible.


Open architecture for developers

The application is Open Source and you can implement your own sync source provider for any storage you want be supported by Synrc Contacts Application. You just need to define how to fetch contacts from sync source, update, delete and add them. Most sync source providers are implemented in one screen of code.

Supports Windows Sync Center

The application registers the Sync Center Partnership and now you can sync all items in one place along with Windows Mobile Devices and Offline Folders.
